# 🛡️ AI-Based Multi Cyber Attack Detection System for Chat and Email Platform

This project is a **real-time AI-powered cybersecurity system** that detects multiple types of cyber attacks such as phishing and malware in chat and email platforms. It uses deep learning models and API-based architecture to provide scalable and automated threat detection.

---

## 🔧 Features

- 🔍 Phishing detection using NLP and URL-based analysis  
- 🦠 Malware detection in file attachments  
- ⚡ Real-time threat detection pipeline  
- 🌐 FastAPI-based modular APIs  
- 📊 Results visualization (confusion matrix, ROC curves)  
- 🧩 Modular architecture for easy scalability
